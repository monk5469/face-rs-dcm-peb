## Code Purpose

This code repository provides MATLAB/SPM12 scripts for reproducing the fMRI preprocessing, first-level GLM, dynamic causal modeling (DCM), parametric empirical Bayes (PEB), Bayesian model comparison, model-adequacy checks, and simulation-based validation analyses used in the accompanying face repetition suppression study.

The code is intended for academic transparency, peer review, and reproducible research related to the manuscript:

Face Repetition Suppression Reflects Condition-Dependent Reconfiguration of Occipitotemporal Effective Connectivity.

## Relationship to Lee et al. (2022)

Parts of the analysis workflow and code structure were adapted and modified from analysis code associated with Lee et al. (2022), which studied effects of face repetition on ventral visual stream connectivity using DCM of fMRI data.

The present repository modifies and extends that prior analysis framework for the current study, including revised preprocessing/GLM organization, alternative input-architecture comparison, condition-specific B-matrix modulation, PEB/BMA inference, family-level B-matrix posterior probabilities, and simulation-based recovery and sensitivity analyses.

Users should cite both the present manuscript/code release and the relevant Lee et al. (2022) work when reusing or discussing analysis components derived from that prior framework.
