# Code Metadata

## Purpose

This repository provides MATLAB/SPM12 scripts for reproducing the fMRI preprocessing, first-level GLM, DCM input-model comparison, DCM-PEB inference, B-matrix family posterior-probability analysis, model-adequacy checks, and simulation-based validation analyses for the face repetition suppression study.

## Manuscript

Face Repetition Suppression Reflects Condition-Dependent Reconfiguration of Occipitotemporal Effective Connectivity.

## Code Origin and Attribution

Some parts of the analysis workflow and code structure were adapted and modified from analysis code associated with Lee et al. (2022), which examined effects of face repetition on ventral visual stream connectivity using DCM of fMRI data.

The present repository extends that prior framework for the current study, including the three-way input-architecture comparison, condition-specific B-matrix analysis, family-level B-matrix posterior probabilities, and simulation-based parameter-recovery and detection-sensitivity analyses.
