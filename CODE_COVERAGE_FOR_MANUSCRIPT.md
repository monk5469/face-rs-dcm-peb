# Code Coverage Against the 20260507 Manuscript

This checklist maps the analyses described in the May 7, 2026 manuscript and supplementary material to the code currently present in this repository.

## Covered by Repository Code

- SPM preprocessing steps described in the manuscript.
  - Dummy-scan removal, realignment, slice-timing correction, coregistration, segmentation, normalization, and 6 mm smoothing.
  - `analysis/scripts/step01_preprocess_fmri.m`
- First-level Imm-Del GLM and subject-level contrasts.
  - `analysis/scripts/step02_first_level_glm.m`
- Optional second-level repeated-measures ANOVA for the nine condition contrasts.
  - `analysis/scripts/step02_first_level_glm.m`
- DCM specification for the two-node rOFA-rFFA model.
  - `analysis/scripts/step03_define_fit_input_models.m`
- Three candidate C-matrix input architectures.
  - Model 1: category-invariant input.
  - Model 2: category-specific face/scrambled input.
  - Model 3: repetition-excluded input.
  - `analysis/utils/build_input_models.m`
- Group-level random-effects BMC from fitted GCM free energies.
  - `analysis/scripts/step03_define_fit_input_models.m`
  - `analysis/scripts/step04_loo_input_model_validation.m`
- Leave-one-out input-model validation.
  - `analysis/scripts/step04_loo_input_model_validation.m`
- Parameter-level PEB/BMR/BMA for condition-specific B-matrix effects.
  - `analysis/scripts/step05_estimate_b_matrix_peb.m`
- Family-level B-matrix posterior probability comparison.
  - Six families: any self-connection, any between-region connection, rOFA self-connection, rFFA self-connection, feedforward rOFA-to-rFFA, and feedback rFFA-to-rOFA.
  - `analysis/scripts/step05_estimate_b_matrix_peb.m`
- Variance-explained model adequacy summary from fitted DCMs.
  - `analysis/scripts/step06_compute_variance_explained.m`
- Monte Carlo parameter-recovery simulation.
  - `analysis/scripts/step07_parameter_recovery_mc.m`
- Simulation-based detection sensitivity.
  - `analysis/scripts/step08_detection_sensitivity_power.m`

## Partially Covered

- Onset extraction from behavioral TSV files.
  - `analysis/scripts/step01_preprocess_fmri.m` can call `correct_stimu_time`.
  - `correct_stimu_time.m` is not included in this repository, so onset files must be provided or that helper must be added to reproduce onset extraction from TSV files.
- ROI localization and VOI extraction.
  - The first-level GLM is covered.
  - AAL-constrained rOFA/rFFA peak search and VOI writing scripts are not included.
- Response-profile and RSI analyses.
  - Variance explained is covered.
  - GLM-on-observed-versus-fitted-signals and RSI plotting scripts described in Supplementary Section S4 are not included.
- Figure generation.
  - Final manuscript figure-generation scripts for Figures 1-6 and Supplementary Figures S1-S7 are not included.

## Implementation Notes

- The 20260507 manuscript compares three input architectures. The standardized code defaults to the three models described in the current manuscript.
- Simulation steps require fitted DCM/PEB/BMA derivatives generated by earlier pipeline steps.
- The helper `fit_peb_model.m` uses `region_spm_dcm_peb_fit` when it is available on the MATLAB path and otherwise falls back to SPM's `spm_dcm_peb_fit` or `spm_dcm_peb`.
